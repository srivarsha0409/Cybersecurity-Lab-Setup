OSI MODEL:
The OSI model has 7 layers:Physical,Data Link,Network,Transport,Session,Presentation,and Application.
It helps standardize network communication.

TCP/IP PROTOCOL SUITE:
TCP/IP is the core communication protocol of the internet.TCp ensures reliable data delivery ,while 
IP handles addressing and routing.

DNS:
DNS(Domain Name System) converts domain names such as google.com into IP addresses.

HTTP and HTTPS:
HTTP is used for web communication on port 80.HTTPS is the secure version of HTTP and uses encryption 
on port 443.

IP Addressing:
An IP address uniquely identofoes a device on a network. Example 192.168.1.10

NAT:
NAT(Network Address Transalation) alows multiple devices to share a single public IP address while 
accessing the Internet.
